# Gossip
Gossip, Java ve JavaFX kullanılarak geliştirilmiş, kullanıcı dostu ve performans odaklı bir masaüstü mesajlaşma uygulamasıdır.

# Öne Çıkan Özellikler
Modern UI Tasarımı: Scene Builder ile oluşturulmuş, kullanıcı deneyimini (UX) ön planda tutan arayüz.

Nesne Yönelimli Mimari: Modüler, sürdürülebilir ve geliştirilebilir Java kod yapısı.

Anlık Etkileşim: Mesaj iletimi ve kullanıcı durum yönetimi süreçlerinin optimizasyonu.

Görsel Dinamizm: Kullanıcı dostu ikonlar ve akıcı geçişler.

# Kullanılan Teknolojiler ve Araçlar
Java: Uygulamanın çekirdek programlama dili.

JavaFX: Zengin grafik arayüz bileşenleri için.

Scene Builder: FXML tabanlı görsel tasarım yönetimi için.

IntelliJ IDEA: Geliştirme ortamı (IDE).
